Les types de données sont des chîanes basiques, des ensembles ordonnées, des listes, des tableau associatives...
Les types de requêtes que l'on peut faire sont: insertion, un update, 

Les types de données : 

 - String : Les différentes requêtes/commandes associées sont SET, GET, INCR, DECR et GETSET.
 - Listes : Les différentes requêtes/commandes associées sont RPUSH, LPUSH, LRANGE, LINDEX et LLEN
 - Hash : Les différentes requêtes/commandes associées sont HSET, HGET, HLEN, HGETALL, HINCRBY, HKEYS, HVALS et HDEL
 - Sets : Les différentes requêtes/commandes associées sont SADD, SCARD, SINTER, SUNION, SDIFF et SINTERSTORE
 - Sets triés : Les différentes requêtes/commandes associées sont ZADD, ZCARD, ZINTER, ZUNION, ZDIFF, ZRANGE, ZRANGEBYSCORE et ZRANK
